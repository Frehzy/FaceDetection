Пожалуйста, отключите антивирусную программу при запуске программы для её корректной работы

Если у программы "сполз" интерфейс, вам необходимо:
1. После установки нажать ПКМ по FaceID.exe, находящийся в папке с программой
2. Нажать на свойства
3. Перейти в "Совместимость"
4. "Изменить параметры для всех пользователей"
5. "Изменить параметры высокого DPI"
6. Поставить галочки у пунктов "Чтобы исправить проблемы масштабирования..." и "Переопределите режим масштабирования высокого разрешения"
7. Нажмите "ОК", "Применить", "ОК", "ОК"

Если программа не работает, выдаёт какие-либо ошибки и т.п., пожалуйста, сообщайте об этом на почту:
